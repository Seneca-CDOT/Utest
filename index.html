Web Usability

Here I present priciples of web usability that I took into consideration while creating this portfolio

  1. The homepage is clearly different from all the other pages on my portfolio.
  2. In the navigation area items are grouped in the way that similar items are next to each other.
  3. There is an obvious way to access the navigation menu on my portfolio, it is in top right corner.
  4. I avoided creating dead end pages on my portfolio because they cause confusion and create additional work for users.
  5. I created a running homepage where you can find links to labs and assignments in addition to navigation menu because scrolling is faster than clicking.
  6. I let users know where they are on the portfolio by using breadcrumbs.
  7. The links on my portfolio tell the user where it leads so the user shouldn't figure this out by himself.
  8. My name is shown in a reasonable size and noticeable location.
  9. I used consistent colors throughout all the pages in my portfolio.
  10. On my homepage, I provided basic information for the user: who am I, what type of website is it, a short information about myself.
